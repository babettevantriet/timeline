De ontwikkeling van de wereldwijde web

1989 - Het donkere design van het webdesign

Het begin van het webdesign stelde het nog weinig voor. Het was vooral heel donker. De kleuren zwart en groen werden vooral veel gebruikt. Er werd nog geen gebruik gemaakt van plaatjes.

1990 - HTML ontstaat

HTML werd bedacht en ontwikkeld door Tim Berners-Lee. Het is een codeer taal die gebruikt kan worden om de content van een website in te plaatsen. Het kan gelezen worden door een browser.

1991 - De eerste website gaat online

De eerste website bestaat uit vrijwel alleen maar hyperlinks, de opmaak van de website is heel basic, dit komt doordat css op dat moment bestaat nog niet. Om de website te bezoeken kunt u hier klikken

1995 - Macromedia Shockwave is ontstaan, en de adds

Shockwave is een script taal die gebruikt wordt door ontwikkelaars, bedoeld om websites toe te staan. HEt heeft als doel om animaties, video's en afbeeldingen te weergeven op een browser. Shockwave word ontwikkeld door Macromedia inc.

1995 - Javascript ontstaat

Html was eerst nog vrij beperkt, maar door de komst van Javascript werd het mogelijk om de gebruiker acties te laten uitvoeren op de pagina. Er zitten wel een paar nadelen aan want javascript moet apart geladen worden van de rest van de pagina.

1996 - CSS ontstaat

Door de komst van CSS, oftewel cascading style sheets, was het mogelijk om de layout van de pagina en de opmaak van de pagina te bepalen. De eerste versie was nog niet zo flexibel en je kon het nog niet gebruiker voor iedere browser.

1998 - Google ontstaat

Dit is tegenwoordig de grootste zoekmachine op het web. Een van de meeste gebruikste websites.

2007 - De eerste mobiele browser

De eerste mobiele browser was safari op apple, hierdoor konden mensen ook op hun telefoon gebruik maken van het web.
